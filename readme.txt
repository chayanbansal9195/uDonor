You'll be promted to log in when you enter the web-app. Depending on the account, you'll be displayed the screen. As a user you can donate wherever you want. You have to choose the source and then what you want to donate and how much. Following this, a delivery boy will pick up your donation within appropriate time.

Technologies used: HTML CSS JavaScript
Frameworks used: NodeJS, Bootstrap, mysql, ejs
Api: Google Map, Waypoints API. Restful APIs
